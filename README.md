该项目是本人的第一个 Vue + TypeScript + Vite + Pinia 项目实践，主要用于 Vue 练手。

- HULITOOL 完全为纯前端项目无后端（但用到了不少 [HotaruApi](https://api.hotaru.icu) 接口），因此网站部署于 GithubPages 服务
- 目前仅有少部分功能仍未实现，稍后也会时不时添加新功能具体查看 [TODO](./TODO.md)
