# HULITOOL
- 该项目是本人的第一个**Vue+TypeScript+Vite+Pinia**项目实战，主要为了练习一下Vue框架开发
- HULITOOL完全为纯前端项目无后端(但用到了不少[HOTARUAPI](https://api.imlolicon.tk)接口)，因此网站通过**GithubPages**服务运行
- 目前仅有少部分功能仍未实现，稍后也会时不时添加新功能，具体查看[CHANGELOG](./CHANGELOG.md)
